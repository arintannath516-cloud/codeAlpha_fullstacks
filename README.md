# CodeAlpha Full Stack Development Tasks

This repository contains complete solutions for the first three tasks from the CodeAlpha Full Stack Development internship assignment.

## Completed Tasks

1. Task 1: Simple E-commerce Store
2. Task 2: Social Media Platform
3. Task 3: Project Management Tool

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Authentication: express-session
- Data storage: LowDB file-based JSON database

## Project Structure

```text
CodeAlpha_FullStackTasks/
- task-1-simple-ecommerce-store/
- task-2-social-media-platform/
- task-3-project-management-tool/
```

## How to Run

Open a terminal inside any task folder and run:

```bash
npm install
npm start
```

Then open the shown localhost URL in your browser.

## Default Behavior

- Every app creates `db.json` automatically on first run.
- Demo data is seeded automatically.
- You can register a new account from each app's signup page.
